รูปแบบการเขียนคำเทศนาใน Google Docs (ใช้รูปแบบนี้เท่านั้น)

1) ชื่อเรื่อง
- ใช้: Heading 1
- ตัวอย่าง: การนมัสการที่แท้จริง

2) หัวข้อหลัก
- ใช้: Heading 2
- ตัวอย่าง: 1. การนมัสการ

3) เนื้อหาอธิบาย
- ใช้: Normal text (Paragraph)
- สามารถ:
  • ตัวหนา
  • ตัวเอียง
  • เปลี่ยนสี
  • ขึ้นบรรทัดใหม่
- ห้าม:
  • ใส่ tag HTML เอง
  • copy จาก Word แบบมีตาราง

4) ข้อพระคัมภีร์ (ถ้ามี)
- ใช้: Italic
- หรือ Normal text + สีเทา

5) เว้นบรรทัด
- ใช้ Enter ปกติ (1–2 ครั้ง)

หมายเหตุ:
- เมื่อเขียนเสร็จ ให้ Publish to web
- ใช้ลิงก์ https://docs.google.com/document/d/ID/edit
